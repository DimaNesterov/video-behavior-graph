# Phase 0 decisions (July 2026)

## Protocol

- Canonical format: (scene_id, frame, timestamp, person_id, x_m, y_m, source), meters, ground plane
- Windows: 8 obs (3.2 s) + 12 pred (4.8 s), annotation step 0.4 s
- Metrics: ADE / FDE, min-over-K=20 for multi-modal models
- Data tracks: obsmat for features/perception (true 0.4 s); Trajectron++ splits for Phase 3 literature comparison only

## Scenes (all verified by overlay + round-trip test)

- Hero scene: zara01 (storefront = object semantics)
- Also: zara02 (flip_px_x), hotel (bench/tram stop), univ (H-old.txt!), eth
- Known caveat: ETH-family benchmark files have a stretched effective timestep (~0.667 s); our obsmat numbers are not literature-comparable (verified numerically)

## Baseline (const velocity, commit 9c07f37)

- zara01 0.58/1.19 (lit. linear ~0.62/1.21 — protocol validated)
- zara02 0.44/0.89 | hotel 0.25/0.47 | univ 0.63/1.24 | eth 0.58/1.16
- This is ablation row A: every model must beat it

## MEVA recon: S3 listing stats (2026-07-05, commit e17c785)

- Corpus: drops-123-r13, 3956 ground-camera clips over 8 days (2018-03-05 .. 2018-03-15),
  4 locations: school (12 cams), bus, hospital, admin. Per camera: ~130-156 clips over 7 days
  (~11-13 h each) -- historical-layer material is sufficient for any candidate,
  so selection is driven by FOV content (map + DIVE), not by volume.
- Full stats: data/raw/meva/camera_stats.csv (generated by scripts/meva_camera_stats.py)
- Anomaly: G474/G475/G476/G479 have ~150 clips but only 0.3-0.5 GB -- consistent with
  thermal IR channels of IR-EO pairs (confirmed on mevadata.org); excluded from candidates.
- Known-defect cameras: G328 (corrupted clip block on 2018-03-05 morning), G639 (was rotated,
  fixed in current drop) -- avoid as finalists.
- Shortlist for DIVE review:
  - bus: G505 (145 clips / 21.6 GB), G340 (145 / 21.1) -- bus stop = natural attractor object,
    wait/stop/board behaviors

    building entrances, richest camera coverage
  - admin: G329 (136 / 18.5) -- building entrance

  person height >= ~50 px; mix of through-traffic and stopping behavior.
